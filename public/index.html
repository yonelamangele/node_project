Users
HTTP Methods Path Description
GET /users/ Gets all the users
GET /users/:id Gets a single user by ID
POST /users/register Adds a user
PATCH /users/update/:id Edits a user info by ID
DELETE /users/delete/:id Deletes a single user by ID
Products
HTTP Methods Path Description

GET /products/:id Gets a single product by ID
POST /products/addProducts Adds a product
PATCH /products/update/:id Edits a product info by ID
